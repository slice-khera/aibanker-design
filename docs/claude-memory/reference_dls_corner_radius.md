## Scale

| Token | Value | Usage |
|-------|-------|-------|
| S | 8px | Chips, tags, tooltips |
| SM | 12px | Snackbars, mid-size containers between S and M |
| M | 16px | Cards, banners, bottom sheet top corners |
| L | 24px | Large containers |
| XL | 32px | AI Banker chip pill, device-frame bezel |
| Pill | 64px | Chips, compact pill elements |
| Circle | 100px | Buttons (pill), avatars, circular elements |

## Corner Smoothing
- Default: 60% (use 100% if platform supports it)
