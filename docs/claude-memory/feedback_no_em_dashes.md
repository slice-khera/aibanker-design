Never use em dashes (-) anywhere. Not in chat replies, not in code comments, not in UI copy, not in .md spec files. Use a period, a comma, a colon, or just two short sentences instead.

**Why:** User has corrected this repeatedly and finds them visually noisy. It is a strong personal style preference, not a one-off note.

**How to apply:** Before writing any output (text or files), scan for `-` and rewrite. Also covers double hyphens used decoratively. En dashes in number ranges like "3–5" are fine.
