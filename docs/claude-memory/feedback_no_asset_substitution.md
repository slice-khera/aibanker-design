When a design asset (illustration, image, etc.) is missing or its URL is dead, do NOT substitute it with random icons, emoji, or placeholder images. Ask the user where to get the actual asset from.

**Why:** User was frustrated by repeated pattern of silently swapping in wrong assets. Illustrations are specific design deliverables, not interchangeable with generic icons.

**How to apply:** If an asset URL 404s or isn't available locally, stop and ask the user. Never assume a fallback is acceptable.
