## Checkbox
- Use for: multiple independent selections
- Touch target: 48x48 | Icon: 24x24
- Unchecked: empty square outline
- Checked: filled square with checkmark, #D30AD7
- Indeterminate: filled with dash
- Disabled: reduced opacity

## Radio
- Use for: single selection from a list
- Touch target: 48x48 | Icon: 24x24
- Empty: circle outline
- Selected: filled circle, #D30AD7
- Disabled: reduced opacity

## Checkmark
- Standalone checkmark (no box): 24x24 in 48px target

## Switch
- Use for: immediate on/off toggles
- Size: 40x24 | Handle: 16x16 white circle | Radius: 100px
- On: track #00A63E, handle right
- Off: track #CDD0D4, handle left
- Disabled: reduced opacity
