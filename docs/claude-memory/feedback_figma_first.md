Use Figma values exactly as specified. Never adjust sizes, spacing, colors, opacity, or any visual property.

**Why:** "Smart" adjustments (bumping icon sizes, tweaking opacity, adjusting spacing by feel) are always wrong and waste time. The user has corrected this repeatedly.

**How to apply:**
- Extract exact values from Figma and use them 1:1
- If something looks off but matches the spec, keep it and flag it to the user
- If the spec is ambiguous, ask - don't guess
- Applies to: padding, margin, gap, font size, icon size, border radius, colors, opacity - everything
