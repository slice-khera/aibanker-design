## Above
- `0px -6px 8px 0px rgba(0,0,0,0.05)`
- Shadow cast upward
- Use for: fixed bottom elements (footer, button group, chat input)

## Below
- `0px 6px 8px 0px rgba(0,0,0,0.05)`
- Shadow cast downward
- Use for: fixed top elements (app bar on scroll, top navigation)
