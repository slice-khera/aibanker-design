## Visualizations
Data displays rendered flat on the chat surface, no bounding box.

| Code type | What it shows |
|-----------|--------------|
| `spend-overview` | Monthly spend amount + line chart trend |
| `category-breakdown` | Horizontal bars per spending category |
| `spend-trend` | Line chart of spending over months |
| `merchant-concentration` | Top merchants bar chart |
| `category-mom` | Month-over-month category comparison bars |
| `spending-heatmap` | Calendar grid of daily spend intensity |
| `payment-mode-donut-v2` | Donut chart of UPI/card/cash split |
| `transaction-table` | List of recent transactions |
| `goal-progress` | Ring/bar showing goal % + status |
| `big-expenses` | Large one-off payments list |

## Widgets
Actionable items enclosed in a container. User can interact/confirm/modify.

| Code type | What it does |
|-----------|-------------|
| `investment-product` | Set up or view an FD/RD - amount selector + activate |
| `add-to-pot` | One-tap "add money to goal" action |
| `savings-plan` | Proposed savings plan - precursor to confirming investment |
| `obligations-list-v2` | Confirm/edit recurring obligations - checkboxes + submit |
