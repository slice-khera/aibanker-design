## Common Usage
- App bar vertical: XS (8px)
- App bar horizontal: S (12px) standard, L (24px) L0
- Content horizontal padding: L (24px)
- Button padding: S/L (12/24px) regular, XS/M (8/16px) small
- Card internal padding: L (24px)
- Bottom sheet content: L (24px) horizontal, M (16px) vertical
- Element gaps: XS (8px), M (16px), L (24px) depending on hierarchy
